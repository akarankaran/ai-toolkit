# .scratch/

Working space for agents and humans. Temp scripts, debug output, intermediate data, scratch
notes, one-off experiments — anything throwaway goes here instead of the repo root or a source
directory.

This directory is **gitignored**: nothing in it is committed. Don't put anything here you want
to keep.

It's working space, not storage. Clean up artifacts once they've served their purpose, and
leave it roughly as empty as you found it.

See `docs/hygiene/README.md` for the full workspace-hygiene rules.
